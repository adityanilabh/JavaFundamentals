1. put(): java.util.HashMap.put() plays role in associating the 
specified value with the specified key in this map. If the map 
previously contained a mapping for the key, the old value is replaced.
  map.put(1,"this is it");

2. get(): java.util.HashMap.get()method returns the value to which the specified
 key is mapped, or null if this map contains no mapping for the key.
 Example: 
     map.get(1);

3.  isEmpty(): java.util.HashMap.isEmpty() method returns true if the map contains
 no key-value mappings

4. size(): java.util.HashMap.size() returns the number of key-value 
mappings in this map.

5. remove(): java.util.HashMap.remove() removes the key-value pair associated 
 with the specified key from the HashMap.

6. containsKey(): java.util.HashMap.containsKey() returns true if the HashMap 
 contains the specified key, otherwise it returns false.

7.containsValue(): java.util.HashMap.containsValue() returns true if the 
HashMap contains the specified value, otherwise it returns false.

8.clear(): java.util.HashMap.clear() removes all the key-value
 pairs from the HashMap.

9.keySet(): java.util.HashMap.keySet() returns a Set containing 
all the keys in the HashMap.

10.values(): java.util.HashMap.values() returns
 a Collection containing all the values in the HashMap
